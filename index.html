How to test:
1. Click the extension icon to open the popup
2. Set "Select count" to 3, enable "Skip first checkbox"
3. Click "Select" — should check 3 items starting from item 2
4. Click "Uncheck" — all checkboxes should be unchecked
5. Click "Invert" — all checkbox states should flip
6. Try toggling "Disable this site" then perform actions
7. Try keyboard shortcuts: Ctrl+Shift+S / U / 9
